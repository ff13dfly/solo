# Mobile Client

内部员工的 AI 交互主入口，移动端优先。

- 本地开发：`http://localhost:9500`

---

## 身份验证

两层验证：

1. **账号登录**：用户名 + 密码（PBKDF2 派生），首次登录后设置本地 PIN 码
2. **PIN 码**：后续启动只需输入 PIN，凭证加密存储在 localStorage

多账号支持，可在账号选择器中切换。

---

## 核心功能

### Chat（对话）
自然语言与 AI 交互，支持文字和图片输入。自动区分任务模式和闲聊模式。

### Focus（意图聚焦）
识别用户意图后进入聚焦模式：
- 动态提取业务参数（仓库、数量、单位等）
- 交互式确认卡片，支持手动修正字段
- 短期记忆（STM）自动保留上下文，无需重复说明

### Form（表单）
结构化业务表单填写。

---

## 技术栈

- React + TypeScript + Vite + Tailwind CSS
- PWA（离线缓存 UI 壳，断网时显示品牌引导页）
- JSON-RPC 2.0（对接 Router）
